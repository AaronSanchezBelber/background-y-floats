## ¿Cómo hacerlo?

1. Crea primero la estructura HTML.
2. Crea un fichero CSS para cada HTML
3. Incluye el fichero CSS dentro de cada HTML

```
<head>
  <link rel="stylesheet" href="styles.css">
</head>
```
